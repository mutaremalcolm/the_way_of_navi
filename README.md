### 👋 Introduction:

---

“The Way of the Na’vi” is a hands-on, form-driven application designed to deepen my understanding of modern frontend development patterns—specifically focusing on **React Hook Form**, **schema validation with Zod**, and **scalable UI architecture using ShadCN and Tailwind CSS**.

Rather than building a generic form app, this project introduces a **cinematic, Avatar-inspired experience**, where users undergo a “rite of passage” through a series of interactive, dynamic, and conditionally validated forms.

The goal of this project is not just to build forms—but to explore how forms can act as **state machines**, driving user experience through validation, interactivity, and real-time feedback.

Built with **Vite + React + TypeScript**, this project emphasizes performance, developer experience, and clean architecture. It also incorporates modern testing practices using **Vitest** and **Testing Library** to ensure reliability and confidence in complex form behavior.

<br/>

### 🔑 Features:

---

#### Core Features:

✔ **Multi-Step Form Experience:** A guided “Toruk Makto Enrollment” journey split into multiple steps, each with its own validation logic and user flow.

✔ **Schema-Based Validation (Zod):** Strongly typed and reusable validation schemas to enforce data integrity and reduce runtime errors.

✔ **React Hook Form Integration:** Efficient form state management with minimal re-renders and powerful control over validation and UI behavior.

✔ **Dynamic Field Arrays:** Users can add and manage multiple entries (e.g., bonds, skills), showcasing advanced RHF capabilities.

✔ **Conditional Validation Logic:** Fields dynamically adapt based on user input (e.g., fear level affecting required inputs).

✔ **Real-Time Form State Feedback:** UI elements respond to `formState` (e.g., `isDirty`, `isValid`, `isSubmitting`) to create an interactive experience.

<br/>

#### Bonus Features:

✔ **ShadCN UI Components:** Accessible, composable UI components built on top of Radix UI primitives.

✔ **Modern Styling with Tailwind CSS:** Clean, responsive design with a focus on glassmorphism and immersive UI effects.

✔ **Form DevTools Integration:** Real-time inspection of form state using React Hook Form DevTools for enhanced debugging and learning.

✔ **Testing with Vitest & Testing Library:** Unit and interaction tests to validate form behavior and user flows.

✔ **State Management with Zustand:** Lightweight global state for managing multi-step form progression and persistence.
